Strumenti utilizzati

  • Tabula: estrazione di dati tabulati da file pdf
  • Libreoffice Calc
  • csv2json: conversione rapida da csv a json
  • QGis

Librerie utilizzate

  • jQuery: framework javascript
  • Nvd3: libreria basata su d3 per la creazione di grafici
  • Leaflet: libreria per la creazione di mappe navigabili
  • Heatcanvas: plugin per leaflet per heatmap
  • Leaflet DVF: plugin per leaflet per data visualization
  • jQuery sparklines: plugin per la generazione di sparkline
  • Topojson: estensione di GeoJson per la codifica topologica delle geometrie
